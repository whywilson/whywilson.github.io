如何制作串口-蓝牙一体式PN532

· 2019-09-14 · # diy # PN532 # MTools

PN532作为史上最低成本RFID的硬件设备，是无法直接上电工作的，它需要在串口通信下传递数据，因此需要串口模块，那么所有串口通信的模块理论上都支持的，本文里接招制作一个带有USB串口和蓝牙串口的一体式PN532。

设备清单

  1. PN532 模块
  2. USB to TTL 模块
  3. 蓝牙串口 模块
  4. 小容量锂电池
  5. USB充电模块
  6. 小开关
  7. 导线若干

连线图

蓝牙串口模块

PN532工作在115200的波特率下，根据你购买的串口模块，通过USB串口模块连接蓝牙模块，

VCC -> 3.3V
GND -> GND
TXD -> RXD
RXD -> TXD

使用AT命令更改波特率为115200，也可以根据实际情况更改模块名称，测试通信正常后，则可以按照连接图连接蓝牙模块。

USB 串口模块

由于PN532自带两对串口输入输出，因此只需要将USB串口与蓝牙串口模块分两侧连接即可，参照连线图连接即可。

电源与充电模块

为了方便，我将开关与电池串联后，接在PN532板上的两个独立孔洞上，然后连接VCC和GND，简直完美的方式
