WooCat - WooCommerce 店群销售报告 iOS客户端

· 2022-03-27 ·

WooCat 是一款可以查看不限数量的 WooCommerce 店铺销售报告的App。WooCat 基于 WooCommerce REST API 和 Store API 进行开发，不需要安装任何插件，仅仅使 WooCommerce 店铺自带特性，即可实现各种便利的功能。

开发简介

WooCat 的iOS端使用 SwiftUI 语言开发，运行系统要求在 iOS15 以上的版本。WooCat 内无任何广告 SDK，所有的数据只储存在设备上与iCloud 中。

更新日志

1.0 初版上线

1.支持添加无限数量店铺
2.支持查看店铺销售、销量、订单数、运费
3.支持天、周、月、年和自定义日期范围
4.多店铺新订单通知
5.支持iCloud数据同步店铺配置

2.0 更新内容

1.支持添加任何 WooCommerce 店铺
2.支持产品跨店铺一键上传
3.支持浏览其他店铺产品信息
4.经典 Cha Ching 新订单通知声
