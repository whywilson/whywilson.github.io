【WooCommerce 插件发布】同步物流信息到 PayPal

· 2021-12-10 · # WooCommerce # ERP # PayPal

作为全球最大的开源电商平台 WooCommerce，虽然拥有众多 ERP 插件，但是没有完善的商品管理、订单处理、物流信息同步等体验。对比 Shopify 的 app，WooCommerce 插件的质量就更显得参差不齐了。在对比了大多数 ERP 后，店小秘可以算得上是同类中做得最完善的，但依旧存在不少问题。

店小秘 ERP 在 WooCommerce 中存在的问题

  1. 无法进行商品采集上架
  2. 点击发货时邮件模板无法匹配物流商名称
  3. 无法同步物流信息到 PayPal
  4. 无法订制邮件模板

本插件功能概要

  1. 修复邮件模板无物流商问题
  2. 当【订单完成】时自动同步物流信息到 PayPal
  3. 手动同步【订单列表】页面物流信息到 PayPal
  4. 手动同步【订单详情】页面物流信息到 PayPal

使用教程

使用环境

  1. WooCommerce 站点
  2. 店小秘 ERP

下载插件

  1. 官方QQ讨论群
  2. 付费下载

安装插件

  1. 进入店铺 Plugins > Add New > Upload Plugin 位置点击选择文件
  2. 选择插件文件 woocommerce-add-tracking-info-to-paypal.zip 并上传
  3. 激活 Add Tracking Info To PayPal 插件

配置

  1. 进入店铺后台 WooCommerce > Settings > PayPal Tracking 位置
  2. 参考 PayPal Developer 官网 配置 Live 环境下的 app
  3. 复制配置 PayPal Client ID 与 Secret 到插件
  4. 勾选 Automate 与 Dianxiaomi Fix 复选框
  5. 保存设置

使用

自动同步物流信息

在店小秘 ERP 中点击发货时，将会自动同步 物流商名称与跟踪号码到 PayPal，由于 PayPal 并不支持所有的物流商名称，若未匹配成功，插件将会用 OTHER 代替物流商信息。
PayPal 支持的物流商名称列表 点击查看

手动同步物流信息

订单列表页面

在订单列表页面将会多出 Courier, Tracking Number 字段和 Synchronize 按钮。

- 已同步
- 未同步
- 同步失败

点击表头或表尾的 Synchronize 按钮将会同步当前页面所有未同步的订单，等待进度消失后将会展示同步结果。

订单详情页面

在订单详情页面将会多出名为 PayPal Tracking 的 Meta Box，其中的按钮的文字将会根据同步状态变化，点击后可以手动同步当前订单的物流信息。

注意事项

  1. 仅状态为 Completed 的订单才可以同步信息到 Paypal
  2. 过多的订单同步会造成服务器内存消耗过多，建议一次同步不要超过50条
  3. 店小秘中不存在的服务商或邮件模板中出现无服务商名称的情况，请 邮件联系我 并提供详细服务商的详细信息。
