使用 OpenAI 生成 WooCommerce 产品 Review

· 2023-07-16 · # OpenAI # ChatGPT # WordPress # WooCommerce

WooCommerce 的产品 Review 是客户主动在产品页留下评价，或在收到货物后对产品信息留下评论。客户留下的 Review 对于店铺来说，可以让客户更加了解产品信息，提升店铺品牌形象。我们有必要为店铺增加足够的 Review 以增加客户或订单的转化率。在本篇博客中，您可以通过 OpenAI 为产品生成无限的真实可信、贴近产品、提升形象的 Review。

OpenAI Tools for WordPress 插件的原理是调用 OpenAI 的 Chat 接口，根据用户提供的关键信息，可以生成不同语言、数量和星级的 Review 信息，包含评论人名、主题、具体内容和日期。

🌐如何安装 OpenAI Tools for WordPress

  1. 在 WordPress 站点后台 Plugins > Add New > 搜索 OpenAI Tools for WordPress ，并安装插件，插件的作者是 MTools Tec.
  2. 安装并激活插件。

🔑如何获取 OpenAI Key

  1. 若您有自己的 OpenAI 账号，且 API Key 可用（免费适用阶段内或付费用户），可以自行创建 API KEY以供 OpenAI Tools for WordPress 插件使用。
  2. 若您没有有效的 OpenAI API Key，我推荐您使用 F2API网站 提供的付费接口服务，它支持支付宝、微信付费，token的费用与OpenAI平台的相当。使用 GPT-3.5-Turbo 模型生成10条 Review 的费用大概是 1毛钱。

✍️添加 OpenAI API Key

将 OpenAPI API Key 添加至 Settings > OpenAI 并提交保存，同时您可以在档期页面看到 API Key 的余额。

🤖生成产品 Review

进入产品编辑页面，展开 OpenAI Reviewer 的 Meta Box, 如果未找到，则在顶部的 Screen Options中启用 OpenAI Reviewer 即可。
生成 Review 前需要提交的信息有：

  1. Review 数量
  2. 开始日期与结束日期
  3. 平均星级
  4. Review主题
  5. 语言
    点击 Submit，等待片刻即可生成相应数量的 Review。

🏭添加 Review 到产品

生成的 Review 信息是可以编辑，例如姓名、星级、日期、评价详情，勾选复选框后，点击底部的 Add Reviews 即可将 Review 添加到产品页面。至此您可以进去产品前台刷新页面查看 Review信息。
