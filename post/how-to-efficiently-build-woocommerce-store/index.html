WooCommerce 自建高效商城搭建方案

· 2022-05-17 · # WordPress # WooCommerce # ERP # PayPal

服务器选择

硬件：Vultr 洛杉矶 2C4G内存 ($24/月)
控制：lnmp 或 宝塔面板

店铺搭建

主题 Flatsome ($59)

Flatsome 在 Themeforest 卖主题，有时有节日折扣，可以关注下官方的 Facebook 群组。

Flatsome主题自带功能

  1. UX Builder 全站定制
  2. 多属性产品 Swatches
  3. 浮动弹窗 Lightbox
  4. UX Block 产品页定制
  5. Catalog 模式不展示价格
  6. 维护模式，404页面等等
  7. 持续的更新

ERP 系统

  1. 店小秘 (免费，仅支持订单处理)
  2. 物流信息自动同步PayPal ($9.99)
    2.1 可以实现在ERP内点击发货，自动同步物流单号到PayPal
    2.2 修复了店小秘的邮件模版中物流商信息丢失问题
  3. 斑马ERP (付费, 支持订单处理，刊登产品，同步物流)

邮件服务

  1. Google Workspace 基础版 ($6/月)
    1.1 域名邮箱服务
    1.2 Google所有相关产品，例如 Drive，Calendar，Keep 等
  2. 邮件服务 WP Mail SMTP (免费)
  3. 邮件模版 Kadence WooCommerce Email Designer (免费)
    可以定制新订单，发货通知，订单完成，添加Note等邮件的模版样式

营销服务

  1. Abandoned Cart Recovery ($59)
  2. Mailchimp (<2000 Contacts 免费)

产品相关

  1. License Manager (免费)
  2. Auto Complete Order (免费)
  3. Customer Reviews for WooCommerce (免费)
    3.1 免费生成所有 Review的集合
    3.2 自动生成 Google Feeds 的 xml 文件

支付网关

  1. WooCommerce PayPal Checkout Gateway (免费)
    优点：个人账号可以开启 PayPal Checkout
    缺点：WooCommerce 官方为了推广 WooCommerce Payment 已停止更新

联系表单

  1. Contact Form 7 (免费)

  2. 人机验证 hCaptcha (免费)

  3. Contact Form 7 Extension For Mailchimp (免费)

网站加速

  1. WP Rocket ($49/年)

  2. Bunny CDN (优惠券白嫖$5.00，可以用很久)

  3. Redis Object Cache (免费)

  4. EWWW Image Optimizer (免费)
    免费版可以使用服务器算力优化图片，付费版使用 EWWW 的服务器优化的。

手机 App

  1. WooCat (完整功能 $9.99，支持多店铺)

  2. WooCommerce (免费，需要安装 JetPack 插件，需要翻墙使用)

机场推荐

目前使用了1年多的 SpeedCat，搭配 ClashX for Windows 和 clashX Pro 在电脑端无阻使用，手机端使用 Clash 和 ShadowRoeckte，比较省心，推荐半年或一年付费，平均不到 ￥20/月，不是最便宜的，但是经过多个对比使用，稳定性很好，可以自动规则代理 ChatGPT。
